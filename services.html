# EduTech – IT Ecosystem Website A fully functional website for EduTech, a virtual tech support startup. Built with HTML, CSS, and JavaScript — no frameworks required. --- ## 📁 File Structure ``` edutech/ ├── index.html ← Homepage (hero, stats, plans, testimonials) ├── services.html ← Services page + AI PC Build Recommender ├── certify.html ← Student certification program page ├── about.html ← About + team + IT infrastructure overview ├── contact.html ← Contact form + FAQ accordion ├── style.css ← All styles (one file) ├── main.js ← All JavaScript (chat AI, animations, form) └── README.md ← This file ``` --- ## 🤖 Free AI Setup (Groq — No Credit Card, No Payment Ever) The live AI chat and PC Build Recommender both use **Groq's free API** — powered by Llama 3.3 70B. ### Step 1: Get Your Free API Key 1. Go to **[console.groq.com](https://console.groq.com)** 2. Click **"Sign Up"** — use your email or Google account 3. **No credit card required at any step** 4. Once logged in, click **"API Keys"** in the left sidebar 5. Click **"Create API Key"**, give it a name, and copy the key (starts with `gsk_...`) > ✅ **Completely free. No credit card. No billing.** Groq's free tier includes 30 requests/minute and 14,400 requests/day — more than enough for a school project. ### Step 2: Add the Key to the Project Open `main.js` and find this line near the top: ```js const GROQ_API_KEY = "YOUR_GROQ_API_KEY_HERE"; ``` Replace it with your actual key: ```js const GROQ_API_KEY = "gsk_YourActualKeyHere"; ``` Save the file. The AI chat widget and PC Recommender will now work instantly. --- ## 🌐 Hosting the Website for Free ### Option A: GitHub Pages (Recommended) 1. Push the project to a GitHub repository (see below) 2. Go to your repo → **Settings** → **Pages** 3. Under "Source", select `main` branch and `/ (root)` folder 4. Click **Save** 5. Your live URL will be: `https://yourusername.github.io/repo-name/` ### Option B: Netlify (Drag & Drop) 1. Go to [netlify.com](https://netlify.com) and create a free account 2. Drag your entire project folder into the Netlify dashboard 3. Get a live URL instantly (e.g., `https://edutech-xyz.netlify.app`) --- ## 🔧 How to Set Up a GitHub Repository ### Step 1: Create a GitHub Account If you don't have one, sign up at [github.com](https://github.com). ### Step 2: Create a New Repository 1.
